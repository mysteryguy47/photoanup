=== ANIMATIONS & INTERACTIONS ===

- On page load: hero text fades up in staggered sequence (eyebrow → title → subtitle → button), each with 0.15s delay between them, opacity 0 to 1 + translateY(20px) to 0, duration 0.8s ease-out

- Gallery images: IntersectionObserver triggers a fade-in + slight upward translate when each image enters viewport, staggered by index
- Section headings: fade up on scroll entry
- Service cards: subtle hover lift
- Navbar: smooth opacity + shadow change on scroll
- Chevron bounce: a gentle up-down CSS keyframe animation on the scroll indicator
- All transitions: duration 0.3s–0.8s, ease curves. NOTHING should feel fast or bouncy. This is a wedding photographer, not a startup.
